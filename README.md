## 🚀 Getting started

1. **Clone the repository**

   ```bash
   git clone <repository-url>
   cd "Web Tech"
   ```

2. **Open a project**

   For plain HTML/CSS/JS, open the relevant `index.html` directly in your browser, or use a live server (e.g. the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) VS Code extension).

   ```bash
   # Example: open a project in the browser
   open path/to/project/index.html
   ```

   *(If a project includes its own build steps or dependencies, see the README inside that project's folder.)*
